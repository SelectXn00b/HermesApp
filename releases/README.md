### Hermes vs AndroidForClaw

| | Hermes | AndroidForClaw |
|--|--------|----------------|
| 包名 | `com.xiaomo.hermes` | `com.xiaomo.androidforclaw` |
| 配置路径 | `/sdcard/.hermes/` | `/sdcard/.androidforclaw/` |
| 可同时安装 | ✅ 互不冲突 | ✅ 互不冲突 |

### Installation

1. Download APK from links above
2. Install on device
3. Configure API Key in Settings or at `/sdcard/.hermes/openclaw.json`
4. Grant Accessibility + Screen Recording permissions
5. Start chatting!

## 📋 System Requirements

- **Android**: 8.0+ (API 26+)
- **Permissions**: Accessibility Service, Display Over Apps, Media Projection
- **Network**: Internet connection for LLM API
